КАК ПРОДОЛЖИТЬ РАБОТУ НА ДРУГОМ УСТРОЙСТВЕ
==========================================

1) Распакуйте этот архив.
2) Установите Node.js (LTS) с https://nodejs.org  — нужен для бэкенда.
3) Откройте папку «nevsky-konditer» в VS Code (File → Open Folder).

ЗАПУСК
- Только клиент (быстро): откройте client/index.html (или Live Server в VS Code).
- Полностью (клиент + сервер + база данных):
      npm run install:all
      npm start
      откройте http://localhost:3000
- Тесты:  npm test
